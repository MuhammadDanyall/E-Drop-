# E-DROP Admin Panel

## 🎯 Overview
Complete website content management system for E-DROP logistics company. Built with HTML, Bootstrap 5, and JavaScript.

## 📁 File Structure
```
admin/
├── login.html              # Admin login page
├── dashboard.html          # Main dashboard with overview
├── pages/                  # Page content management
│   ├── home.html          # Home page content editor
│   ├── about.html         # About page content editor
│   ├── services.html      # Services page content editor
│   └── contact.html       # Contact page content editor
├── services/               # Services management
│   ├── view.html          # Services list & management
│   ├── add.html           # Add new service
│   └── edit.html          # Edit existing service
├── messages.html           # Contact form messages
├── media.html             # Image/file management
├── profile.html           # Admin profile settings
└── README.md              # This file
```

## 🔧 How to Use

1. **Start with Login:**
   - Open `admin/login.html`
   - Use credentials: admin@edrop.com / admin123

2. **Navigate Dashboard:**
   - View website statistics
   - Access different management sections

3. **Manage Content:**
   - Use Pages section to edit website text
   - Use Services Management for service control
   - Check Messages for customer inquiries

4. **Upload Media:**
   - Go to Media section
   - Drag & drop or click to upload images
